# API Endpoints

## HTML API

### Root

- `GET /`

## JSON API

### Users

- `POST /api/users`
- `PATCH /api/users`

### Session

- `GET /api/session`
- `POST /api/session`
- `DELETE /api/session`

### Messages

- `GET /api/messages`
- `POST /api/messages`
- `GET /api/messages/:id`

### Fora

- `GET /api/fora`
- `POST /api/channels`
- `GET /api/fora/:id`
- `GET /api/channels/:id/messages`
